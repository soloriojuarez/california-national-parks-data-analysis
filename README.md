# US National Park Visitation Analysis

This project analyzes visitation data from U.S. National Parks to determine which parks are the most popular.

## Tools Used
- Wolfram Mathematica
- Data Visualization
- Data Analysis

## Dataset
U.S. National Park visitation dataset including visitor counts, park budgets, and staffing levels.

## Project Goal
Identify trends in national park popularity using a large dataset.

## Files
- Dataset (.csv)
- Mathematica analysis notebook
- Project presentation slides
